## Usage

To use the component, import the `Tooltip` component from the package:

```jsx
import { Tooltip } from "@syntatis/kubrick";
```

Then, you can use it in your application by adding the `Tooltip` component and passing the `content` prop with the content you want to display on the tooltip. You will also need to pass a component as a child to the `Tooltip` component, which will be the element that will trigger the tooltip. The child component should be

```jsx
<Tooltip content="">
	<button>Hover me</button>
</Tooltip>
```
